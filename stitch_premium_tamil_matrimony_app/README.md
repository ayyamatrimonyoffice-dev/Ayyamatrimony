# Stitch HTML Mockups (reference only)

This folder contains the **original Google Stitch HTML designs**.

## Run the app here instead

The React Native app lives in:

```
../matrimony-app
```

```powershell
cd "..\matrimony-app"
npm start
```

Then press `w` for web, or open http://localhost:8081

Do **not** run `npx expo start` from this folder — it will fail with a 500 / MIME type error because there is no Expo app here.
